# 📝 Todo App

> ✅ Ứng dụng quản lý công việc đơn giản, giúp bạn theo dõi và hoàn thành task mỗi ngày.

🌐 **Live Demo:** https://panbap.github.io/Todo/

---

## 🚀 Giới thiệu

Đây là một project **Todo List (To-do App)** được xây dựng bằng JavaScript thuần, giúp bạn quản lý công việc một cách trực quan và hiệu quả.

Phù hợp để:
- 🧑‍💻 Luyện tập JavaScript  
- 📚 Hiểu DOM & Event  
- 🎯 Làm mini project front-end
